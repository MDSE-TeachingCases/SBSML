# Lab 1 - Metamodeling

In this assignment the students have to develop a metamodel for the **Smart Building System Modeling Language (SBSML)** using Ecore and implement constraints for that metamodel using OCL. The full description can be found in the [assignment](./assignment.pdf). There are no template projects, so the students have to start from scratch with a new Eclipse Ecore Modling Project. The [sample solution](./sample_solution) will used for [lab 2](../lab2), [lab 3](../lab3) and [lab 4](../lab4).
